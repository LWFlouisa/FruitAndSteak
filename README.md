# FruitAndSteak
A 4D array in C.

## Concept
~~~
  //               0
  //          apples bananas kiwis
  // apples   a,a    a,b     a,k
  // bananas  b,a    b,b     b,k
  // kiwis    k,a    k,b     k,k

  //               1
  //          steak chicken fish
  // steak    s,s   s,c     s,f
  // chicken  c,s   c,c     c,f
  // fish     f,s   f,c     f,f
~~~
